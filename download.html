How to Install

  1. 下载 DMG 文件
  2. Download the DMG file
  3. 打开后将 EdgeTalk 拖到应用程序文件夹
  4. Open it and drag EdgeTalk to Applications
  5. 从应用程序文件夹启动（首次需右键 → 打开）
  6. Launch from Applications (right-click → Open on first launch)
  7. 根据引导授予麦克风和辅助功能权限
  8. Grant microphone and accessibility permissions when prompted
